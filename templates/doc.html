Mamba-GNN 项目操作手册

版本 v3.0.0 | 济南网约车数据专用版

1. 系统简介

本系统是一个基于 Mamba-GNN 的智能网约车需求预测平台。系统基于 济南市公共数据开放网 的真实网约车订单数据，通过数据合并、清洗与筛选，构建高精度的时空预测模型，并通过 Web 界面提供实时可视化服务。

2. 环境检测 (Environment)

在开始数据处理前，必须确保 Python 环境及相关依赖（Torch, Flask, Mamba-SSM 等）已正确安装。

python env.py

3. 数据处理流水线 (Data Pipeline)

本版本采用了新的数据源处理方式，请严格按照以下步骤操作。

Step 3.1 数据采集 (Data Acquisition)

前往 济南市公共数据开放网，下载网约车订单信息压缩包。
操作： 解压下载的压缩包，将获得的多个 .csv 原始文件放置于项目根目录或指定文件夹下。

Step 3.2 数据合并 (Data Merging)

将解压后的分散 CSV 文件合并为一个总的原始数据文件。

python merge.py

✓ 输出结果： 生成文件 data/jinan_taxi.csv，包含所有原始订单记录。

Step 3.3 数据集终处理 (Final Processing)

对合并后的数据进行初步清洗，处理格式错误或冗余数据，生成最终用于模型训练的高质量数据集。 此步骤至关重要，直接决定模型效果。

python final.py

✓ 输出结果： 生成文件 data/finaldata.csv。

4. 业务分析与模型训练

Step 4.1 业务数据分析 (EDA)

对处理好的最终数据集进行探索性分析，生成统计图表，帮助理解数据分布特征。

python analysis.py

Step 4.2 模型训练 (Training)

读取 data/finaldata.csv，构建图神经网络结构并使用 Mamba 模块进行时序特征提取。 训练完成后会自动保存最优权重文件。

python trainmodel.py

✓ 输出结果： 生成模型权重文件（如 best_model_Mamba-GNN.pt）。

5. 可视化部署 (Visualization)

Step 5.1 安装所需js库

下载好chart.js和tailwindcss.js，便于后端的本地可视化

python config.py

Step 5.2 启动可视化服务

加载训练好的模型，启动 Flask 服务器。系统将提供交互式的网页界面展示预测结果。

python app.py

✓ 成功启动后，浏览器访问: http://127.0.0.1:8080
